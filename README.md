# Sorriso & Confiança - Landing Page Odontológica

Este projeto é uma Landing Page de alta conversão desenvolvida para uma clínica de odontologia, focada em transmitir autoridade, higiene e acolhimento humano.

## 🚀 Tecnologias Utilizadas

- **HTML5 Semântico**: Estrutura robusta e amigável para SEO.
- **Tailwind CSS (CDN)**: Estilização moderna, responsiva e de alta performance.
- **Google Fonts**: Combinação tipográfica entre *Playfair Display* (Serif) e *Inter* (Sans-serif).
- **Phosphor Icons**: Ícones minimalistas para uma interface clean.

## 🎨 Design System

- **Paleta de Cores**: Baseada em *Azul Serenity*, Branco e *Cinza Pérola*.
- **Acessibilidade**: Contraste adequado e navegação fluida.
- **Responsividade**: Design mobile-first impecável.

## 🛠️ Como executar o projeto

Como o projeto utiliza Tailwind via CDN, não há necessidade de instalação de dependências pesadas para visualização local:

1. Clone o repositório:
   ```bash
   git clone https://github.com/ogabrielavelino/P2-odonto.git
   ```
2. Abra o arquivo `index.html` em qualquer navegador moderno.
